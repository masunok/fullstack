# AIZEVA 디자인 문서

### 디자인 가이드라인
- 반응형 완벽 지원 (puppeteer MCP 로 반응형 디자인도 문제가 없는지 확인)
- https://tx.shadcn.com/ 스타일로 전체 웹페이지에 일관된 디자인 적용

## XML 디자인 파일 개요

- AIZEVA 서비스의 모든 페이지 구조를 XML 형식으로 정의합니다. 
- 각 XML 파일은 페이지의 구조와 요소를 명확하게 표현하여 구현 시 참조할 수 있도록 합니다.

## XML 파일 목록

### 1. 공통 컴포넌트
- **@header.xml**: 모든 페이지 공통 헤더
- **@footer.xml**: 모든 페이지 공통 푸터

### 2. 메인 및 인증 페이지
- **@main.xml**: 메인 페이지 (/) - 최신 게시글 표시
- **@login.xml**: 로그인 페이지 (/login)
- **@signup.xml**: 회원가입 페이지 (/signup)

### 3. 게시판 및 게시글 페이지
- **@board_list.xml**: 게시판 목록 페이지
- **@board_detail.xml**: 게시판 상세 (게시글 목록)
- **@post_list.xml**: 게시글 목록 (페이지네이션 포함)
- **@post_detail.xml**: 게시글 상세 (댓글 포함)
- **@post_write.xml**: 게시글 작성/수정

### 4. 관리자 페이지
- **@admin_users.xml**: 사용자 관리 페이지
- **@admin_boards.xml**: 게시판 관리 페이지

## 반응형 디자인 요구사항

### 모바일 (768px 이하)
- 햄버거 메뉴로 네비게이션 숨김
- 게시글 목록: 카드 형태로 세로 배치
- 페이지네이션: 간소화된 이전/다음 버튼

### 태블릿 (768px - 1024px)
- 사이드바 일부 축소
- 게시글 목록: 2단 그리드

### 데스크톱 (1024px 이상)
- 전체 사이드바 표시
- 게시글 목록: 테이블 형태

## UI 컴포넌트 규칙

### 페이지네이션
- 10개씩 표시
- 버튼: [처음] [이전] [1] [2] [3] [다음] [끝]
- 현재 페이지 강조 표시

### 폼 검증
- 실시간 검증 피드백
- 에러 메시지는 해당 필드 하단에 표시
- 비밀번호: 영문자+숫자+특수문자 10자리 이상

### 검색 기능
- 검색 범위: 전체/특정게시판 선택
- 검색 대상: 제목+내용 통합 검색
- 검색 결과: 하이라이트 표시
